MySQL Setup Instructions

Follow these steps to set up MySQL for the Project Monitoring System:

Step 1: Start MySQL Server

If you're using XAMPP:

  1. Open XAMPP Control Panel
  2. Find the "MySQL" row
  3. Click the "Start" button
  4. Wait until the status shows "Running"

Alternatively, you can run the provided batch file:

start-mysql.bat

Step 2: Create Database

Open a command prompt or terminal and run:

mysql -u root -e "CREATE DATABASE IF NOT EXISTS project_monitoring;"

Step 3: Execute Database Schema

Navigate to the project directory and run:

mysql -u root project_monitoring < mysql-schema.sql

Step 4: Verify Installation

You can verify that everything is set up correctly by checking that the tables were created:

mysql -u root project_monitoring -e "SHOW TABLES;"

You should see the following tables:

  • districts
  • document_types
  • documents
  • progress_reports
  • project_sections
  • projects
  • trade_progress
  • trades
  • user_roles
  • users

Success!

Once you've completed these steps, you can start the development server:

npm run dev

Then visit http://localhost:3000 to access the application.

Note:

If you encounter any issues:

  • Make sure no other MySQL instances are running
  • Check that the MySQL service is properly started
  • Ensure you have the necessary permissions to create databases
  • Verify that the MySQL bin directory is in your system PATH
